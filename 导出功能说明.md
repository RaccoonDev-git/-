# 📤 导出功能说明文档

## 概述
本系统提供完善的数据导出功能，支持多种数据类型和导出格式。本文档详细说明所有可用的导出功能及其使用方法。

---

## 🎯 导出功能分类

### 📊 **1. 学生信息导出**

#### 1.1 导出所有学生 ⭐
- **API端点**: `GET /api/admin/students/export`
- **权限要求**: ADMIN 或 TEACHER
- **文件格式**: Excel (.xlsx)
- **文件名**: `students.xlsx`
- **包含字段**:
  - 学号
  - 姓名
  - 年级
  - 班级
  - 专业
  - 手机号
  - 邮箱
  - 备注

**前端操作**:
```
教师页面 → 学生管理 → 点击"导出数据"按钮
```

**API调用示例**:
```javascript
GET http://localhost:8080/api/admin/students/export
Headers:
  Authorization: Bearer {token}
```

---

#### 1.2 导出选中学生 ⭐
- **API端点**: `POST /api/admin/students/export/selected`
- **权限要求**: ADMIN 或 TEACHER
- **文件格式**: Excel (.xlsx)
- **文件名**: `selected_students.xlsx`
- **包含字段**: 与"导出所有学生"相同

**请求体**:
```json
{
  "studentIds": ["2023001", "2023002", "2023003"]
}
```

**前端操作**:
```
教师页面 → 学生管理 → 
进入批量操作模式 → 选择学生 → 点击"导出选中"按钮
```

**API调用示例**:
```javascript
POST http://localhost:8080/api/admin/students/export/selected
Headers:
  Authorization: Bearer {token}
  Content-Type: application/json
Body:
  {
    "studentIds": ["2023001", "2023002"]
  }
```

---


```javascript
// 导出所有学生

  const response = await fetch('http://localhost:8080/api/admin/students/export', {

  });
  const blob = await response.blob();
  downloadBlob(blob, 'students.xlsx');

  const response = await fetch('http://localhost:8080/api/admin/students/export/selected', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'

  });
  const blob = await response.blob();
  downloadBlob(blob, 'selected_students.xlsx');

    headers: { 'Authorization': `Bearer ${token}` }

  downloadBlob(blob, `student_${studentId}.xlsx`);
```

## 📝 使用示例

### 示例 1: 导出所有学生（JavaScript）
```javascript
async function exportAllStudents() {
  try {
    showNotification('正在准备导出...', 'info');
    
    const token = localStorage.getItem('token');
    const response = await fetch('http://localhost:8080/api/admin/students/export', {
      method: 'GET',
      headers: {
        'Authorization': `Bearer ${token}`
      }
    });

    if (!response.ok) {
      throw new Error('导出失败');
    }

    const blob = await response.blob();
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = `students_${new Date().toISOString().split('T')[0]}.xlsx`;
    a.click();
    URL.revokeObjectURL(url);
    
    showNotification('导出成功', 'success');
  } catch (error) {
    showNotification(`导出失败: ${error.message}`, 'error');
  }
}
```


## ⚠️ 注意事项

### 1. 性能考虑
- **大数据量**: 导出大量数据时可能需要较长时间
- **建议**: 使用分页或筛选条件限制导出数据量
- **超时设置**: 建议设置较长的请求超时时间（如60秒）

### 2. 文件格式
- **Excel格式**: 适合结构化数据，支持样式和格式
- **CSV格式**: 适合大量数据，文件更小，兼容性好
- **字符编码**: 所有文件使用UTF-8编码

### 3. 安全考虑
- **权限验证**: 所有导出请求都需要JWT认证
- **数据脱敏**: 敏感信息应考虑脱敏处理
- **审计日志**: 导出操作应记录审计日志

### 4. 错误处理
```javascript
// 推荐的错误处理方式
async function safeExport(url, filename) {
  try {
    const response = await fetch(url, {
      headers: { 'Authorization': `Bearer ${token}` }
    });
    
    if (!response.ok) {
      if (response.status === 401) {
        throw new Error('未授权，请重新登录');
      } else if (response.status === 403) {
        throw new Error('权限不足');
      } else if (response.status === 404) {
        throw new Error('数据不存在');
      } else {
        throw new Error(`导出失败: ${response.statusText}`);
      }
    }
    
    const blob = await response.blob();
    downloadBlob(blob, filename);
    return true;
  } catch (error) {
    console.error('导出错误:', error);
    showNotification(error.message, 'error');
    return false;
  }
}
```

## 📞 常见问题 (FAQ)

### Q1: 导出文件为什么是乱码？
**A**: 确保文件使用UTF-8编码打开。对于CSV文件，可以用Excel的"数据"->"从文本"功能导入，并选择UTF-8编码。

### Q2: 导出的Excel文件打不开？
**A**: 检查文件是否完整下载，文件大小是否正常。如果文件损坏，尝试重新导出。

### Q3: 导出时提示"权限不足"？
**A**: 检查：
1. 是否已登录
2. JWT Token是否过期
3. 当前用户角色是否有导出权限

### Q4: 如何导出特定条件的数据？
**A**: 目前支持的筛选方式：
- 按学生ID导出（单个或批量）
- 按课程ID导出
- 如需更复杂的筛选，可先在前端筛选后再导出选中数据

### Q5: 导出速度慢怎么办？
**A**: 
1. 减少导出数据量
2. 使用批量导出而非全部导出
3. 避免在系统高峰期导出
4. 联系管理员检查服务器性能
