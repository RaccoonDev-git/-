# 用户批量导入Excel模板

## Excel文件格式要求

### 文件格式
- 支持: `.xlsx` 和 `.xls` 格式
- 编码: UTF-8

### 表格结构

| 列号 | 列名 | 说明 | 必填 | 示例 |
|-----|------|------|------|------|
| A | 用户名 | 唯一标识,3-20字符 | 是 | student001 |
| B | 密码 | 至少6位字符 | 是 | password123 |
| C | 角色 | STUDENT/TEACHER/ADMIN | 是 | STUDENT |
| D | 邮箱 | 有效邮箱地址 | 否 | student@example.com |
| E | 手机 | 手机号码 | 否 | 13800138000 |

### 示例数据

```
用户名      | 密码        | 角色      | 邮箱                  | 手机
-----------|------------|----------|----------------------|-------------
student001 | pass123456 | STUDENT  | student1@test.com    | 13800001111
student002 | pass123456 | STUDENT  | student2@test.com    | 13800002222
teacher001 | pass123456 | TEACHER  | teacher1@test.com    | 13800003333
admin001   | pass123456 | ADMIN    | admin1@test.com      | 13800004444
```

## 导入规则

### 验证规则
1. **用户名**: 必填,不能重复
2. **密码**: 必填,长度至少6位
3. **角色**: 必填,仅支持: STUDENT, TEACHER, ADMIN
4. **邮箱**: 选填,格式验证
5. **手机**: 选填

### 错误处理
- 第一行作为标题行,自动跳过
- 空行自动跳过
- 单行错误不影响其他行导入
- 导入完成后显示成功/失败统计

## 使用方法

### 1. 准备Excel文件
1. 使用Excel/WPS打开新文件
2. 第一行填写列标题
3. 从第二行开始填写用户数据
4. 保存为`.xlsx`或`.xls`格式

### 2. 执行导入
1. 登录管理员账号
2. 进入"用户管理"页面
3. 点击"批量导入"按钮
4. 选择准备好的Excel文件
5. 等待导入完成
6. 查看导入结果统计

### 3. 查看结果
导入完成后会显示:
- 总行数
- 成功数量
- 失败数量
- 详细错误信息(如有)

## 注意事项

⚠️ **重要提醒**:
1. 用户名不能与现有用户重复
2. 角色名称必须大写(STUDENT/TEACHER/ADMIN)
3. 建议先小批量测试导入
4. 导入前建议备份数据库
5. 导入的用户默认状态为ACTIVE(活跃)

## 常见问题

**Q: 导入失败怎么办?**
A: 查看错误详情,根据提示修改Excel文件后重新导入。

**Q: 部分成功部分失败怎么办?**
A: 成功的用户已导入,修改失败行的数据后重新导入失败的用户。

**Q: 可以导入多少用户?**
A: 建议单次导入不超过1000条,大批量数据可分多次导入。

**Q: 如何批量修改用户信息?**
A: 目前仅支持批量导入新用户,修改需在用户管理页面逐个操作。

## API接口

### 批量导入API
```
POST /api/admin/users/import
Content-Type: multipart/form-data
Authorization: Bearer {token}

参数:
- file: Excel文件(MultipartFile)

响应:
{
  "success": true,
  "successCount": 10,
  "failCount": 2,
  "totalRows": 12,
  "errors": [
    "第3行: 用户名已存在",
    "第7行: 角色值无效"
  ]
}
```
