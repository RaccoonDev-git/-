# 📤 多格式导出功能 - 快速参考

## 🎯 一句话总结

**支持 CSV、Excel、JSON 三种格式导出学生数据，可选择导出范围和字段**

---

## ⚡ 快速使用

### 基本步骤

```
点击"导出数据" → 选择范围 → 选择格式 → 选择字段 → 点击"导出"
```

### 5 秒上手

1. 点击工具栏的 **"导出数据"** 按钮
2. 选择想要的 **格式**（CSV/Excel/JSON）
3. 点击 **"导出"** 按钮
4. 文件自动下载到浏览器下载文件夹

---

## 📊 格式对照表

| 格式      | 文件扩展名 | 最佳用途           | 打开方式        |
| --------- | ---------- | ------------------ | --------------- |
| **CSV**   | `.csv`     | 表格编辑、数据交换 | Excel, Numbers  |
| **Excel** | `.xlsx`    | 数据分析、格式化   | Microsoft Excel |
| **JSON**  | `.json`    | 程序处理、API 集成 | 文本编辑器      |

---

## 🎛️ 导出选项速查

### 导出范围

- **全部学生** 🌐 - 所有数据
- **当前筛选结果** 🔍 - 筛选后的数据
- **选中的学生** ✅ - 批量选中的数据

### 可导出字段

✅ 姓名 | ✅ 学号 | ✅ 年级 | ✅ 班级 | ✅ 专业 | ✅ 联系方式

---

## 💡 常用场景

### 场景 1：备份所有学生数据

```
范围: 全部学生
格式: Excel
字段: 全选
```

### 场景 2：导出特定班级名单

```
1. 先筛选班级（如：19软工A1）
2. 范围: 当前筛选结果
3. 格式: CSV
4. 字段: 姓名、学号
```

### 场景 3：导出联系人信息

```
1. 启用批量操作，选中学生
2. 范围: 选中的学生
3. 格式: CSV
4. 字段: 姓名、联系方式
```

---

## 🚨 故障速解

| 问题           | 解决方案                      |
| -------------- | ----------------------------- |
| Excel 导出失败 | F12 检查 SheetJS 库是否加载   |
| CSV 文件乱码   | 已自动处理，用 Excel 打开即可 |
| 导出 0 条记录  | 检查是否有学生数据或筛选条件  |
| 选中学生不可用 | 先启用"批量操作"模式          |

## 🔍 文件命名规则

```
学生数据_2025-10-08T14-30-25.csv
学生数据_2025-10-08T14-30-25.xlsx
学生数据_2025-10-08T14-30-25.json
```

**格式**: `学生数据_日期时间.扩展名`

## ⌨️ 快捷操作

- **全选字段**: 点击"全选"按钮
- **取消全选**: 点击"取消全选"按钮
- **关闭对话框**: 点击右上角 × 或按 ESC 键
- **快速导出**: 保持默认设置直接点"导出"
