# 课程数据转换为学生导入格式 - 使用指南

## 📋 问题描述

当你从学习平台（如超星学习通）导出的课程数据（包含学生完成情况）需要导入到学生管理系统时，由于格式不匹配会显示"没有数据"。

### 原始文件格式示例

```csv
任务点完成详情,...
课程：编译原理...
学生姓名,学号/工号,学校,院系,专业,班级,编译系统简介,词法分析器的设计...
沈罡,20181112737,上海第二工业大学,计算机与信息工程学院,软件工程,18软工A1,...
```

### 需要的格式

```csv
姓名,学号,年级,班级,专业,手机号,备注
沈罡,20181112737,2018,18软工A1,软件工程,,从课程数据导入 2025-10-08
```

## 🔧 解决方案

使用 `convert-course-data.py` 脚本自动转换格式。

## 📖 使用步骤

### 1. 准备文件

确保你有从学习平台导出的课程数据文件（CSV 或 Excel）。

### 2. 运行转换脚本

#### 方法一：自动命名输出文件

```powershell
python convert-course-data.py "你的课程数据文件.csv"
```

输出文件会自动命名为：`你的课程数据文件_学生导入格式.csv`

#### 方法二：指定输出文件名

```powershell
python convert-course-data.py "你的课程数据文件.csv" "学生信息.csv"
```

### 3. 查看转换结果

脚本会显示：

- ✓ 成功提取的学生数量
- 输出文件路径
- 前 3 条学生数据预览

### 4. 导入到系统

1. 登录系统，进入"学生管理"页面
2. 点击"批量导入"按钮
3. 选择转换后的 CSV 文件（`*_学生导入格式.csv`）
4. 预览数据，确认无误后点击"开始导入"

## 🎯 脚本功能特点

### ✅ 自动识别

- 自动检测文件编码（UTF-8、GBK、GB2312 等）
- 智能查找包含学生信息的标题行
- 自动识别"姓名"、"学号"、"专业"、"班级"等列

### ✅ 数据提取

- 提取学生姓名、学号
- 从学号自动推断年级（前 4 位）
- 提取专业、班级信息
- 自动添加导入日期备注

### ✅ 格式标准化

- 输出标准的学生导入格式
- 使用 UTF-8 with BOM 编码（兼容 Excel）
- 符合系统导入模板要求

### ✅ 数据验证

- 跳过空行和无效数据
- 过滤标题行重复
- 确保必需字段完整

## ⚠️ 注意事项

### 1. 文件要求

- 必须包含"学生姓名"和"学号/工号"列
- 文件编码建议为 UTF-8 或 GBK
- 支持 CSV 和 Excel (xlsx) 格式

### 2. 数据处理

- 手机号字段为空（原始数据中通常没有）
- 年级从学号前 4 位提取（如：20191112403 → 2019）
- 备注自动添加导入日期

### 3. 常见问题

**问题 1**：提示"未找到必需的列"

- **原因**：文件格式不符合预期
- **解决**：检查文件是否包含"学生姓名"和"学号"列

**问题 2**：转换后数据为空

- **原因**：标题行识别错误
- **解决**：手动检查 CSV 文件，确保标题行格式正确

**问题 3**：导入时显示"没有数据"

- **原因**：未使用转换后的文件，或文件编码问题
- **解决**：
  1. 确认使用的是 `*_学生导入格式.csv` 文件
  2. 文件应为 UTF-8 编码
  3. 刷新浏览器缓存（Ctrl+F5）

## 🔍 验证转换结果

```powershell

# 统计学生数量（减去表头）

```

### 使用 Excel 打开

1. 右键点击 CSV 文件 → 使用 Excel 打开
2. 如果出现乱码，选择"数据" → "从文本/CSV" → 选择"UTF-8"编码
3. 确认数据显示正确

## 🛠️ 技术细节

### 支持的文件编码

- UTF-8 (with/without BOM)
- GBK
- GB2312
- GB18030

### 提取的字段映射

| 原始字段  | 转换字段 | 说明                  |
| --------- | -------- | --------------------- |
| 学生姓名  | 姓名     | 必需                  |
| 学号/工号 | 学号     | 必需，用于唯一标识    |
| 专业      | 专业     | 可选                  |
| 班级      | 班级     | 可选                  |
| -         | 年级     | 从学号前 4 位自动提取 |
| -         | 手机号   | 留空                  |
| -         | 备注     | 自动添加导入日期      |
